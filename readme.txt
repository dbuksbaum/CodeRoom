CodeRoom v0.1.5 Development

Keyboard commands:
Ctrl + S:       Save
Ctrl + W:       Save As
Ctrl + O:       Open
Ctrl + N:       New Document
Ctrl + Q:       Quit
Escape:         Quit
Ctrl + H:       Choose Syntax Highlighting
Ctrl + Return:  Create new line after the current (same as End + Return)
F5 to F12:      QuickInsert, as defined in syntax file
